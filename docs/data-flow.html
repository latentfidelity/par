⚡ Pipeline Steps

Step 1
MCP Request
Agent sends a tool call via Streamable HTTP or SSE transport
Step 2
Tool Dispatch
Gateway validates params with Zod and routes to the correct handler
Step 3
Storage I/O
Flat-file JSON reads/writes to the Docker-mounted /data/meta volume
Step 4
Embedding
MiniLM-L6-v2 generates 384-dim vectors for semantic search indexing
Step 5
KG Extraction
Entity-relationship triples extracted and merged into the knowledge graph
Step 6
Response
Structured MCP result returned to the agent with metadata and content
